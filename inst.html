Инструкция по установке v2ray на OpenWRT

Шаг 1: Установка ключа v2raya

Выполните следующую команду для загрузки и установки ключа v2raya:

wget http://downloads.sourceforge.net/project/v2raya/openwrt/v2raya.pub -O /etc/opkg/keys/94cc2a834fb0aa03

Шаг 2: Добавление репозитория v2raya

Добавьте репозиторий v2raya с помощью следующей команды:

echo "src/gz v2raya https://downloads.sourceforge.net/project/v2raya/openwrt/$(. /etc/openwrt_release && echo "$DISTRIB_ARCH")" | tee -a "/etc/opkg/customfeeds.conf"

Шаг 3: Обновление списка пакетов

Обновите список пакетов с помощью следующей команды:

opkg update

Шаг 4: Обновление установленных пакетов

Обновите установленные пакеты с помощью следующей команды:

opkg list-upgradable | cut -f 1 -d " " | xargs opkg upgrade

Шаг 5: Установка необходимых пакетов

Установите необходимые пакеты с помощью следующей команды:

opkg install luci-i18n-base-ru v2raya xray-core kmod-nft-tproxy iptables-nft luci-app-v2raya

Шаг 6: Настройка v2raya

Настройте v2raya с помощью следующих команд:

uci set v2raya.config.enabled='1'
uci commit v2raya

Шаг 7: Запуск v2raya

Запустите v2raya с помощью следующих команд:

/etc/init.d/v2raya enable
/etc/init.d/v2raya start

Шаг 8: Загрузка баз данных GeoIP и GeoSite

Загрузите актуальные базы данных GeoIP и GeoSite с помощью следующих команд:

wget -O /usr/share/xray/geoip.dat https://github.com/1andrevich/Re-filter-lists/releases/latest/download/geoip.dat
wget -O /usr/share/xray/geosite.dat https://github.com/1andrevich/Re-filter-lists/releases/latest/download/geosite.dat

Шаг 9: Настройка крон для обновления баз данных

Добавьте следующие строки в крон для автоматического обновления баз данных GeoIP и GeoSite:

echo "0 5 * * * wget -O /usr/share/xray/geoip.dat https://github.com/1andrevich/Re-filter-lists/releases/latest/download/geoip.dat" >> /etc/crontabs/root
echo "0 5 * * * wget -O /usr/share/xray/geosite.dat https://github.com/1andrevich/Re-filter-lists/releases/latest/download/geosite.dat" >> /etc/crontabs/root

Теперь v2ray и v2raya установлены и настроены на вашем устройстве с OpenWRT.
